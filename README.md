# 算数そうふくしゅうアプリ

小学1年生〜3年生向けの算数総復習Webアプリです。

子どもが紙を使わず、スマホ・タブレット・PCで、選択式の問題を解きながら復習できることを目指しています。

## 概要

このアプリでは、小学1年生〜3年生の算数を学年・単元ごとに練習できます。

- すべて選択式で解ける
- サクサク解く10問モードがある
- じっくり考える5問モードがある
- 文章題や筆算をステップ式で確認できる
- 間違えた問題を保存し、苦手こくふくで再挑戦できる
- 保護者が学習状況を確認できる
- メモボード、効果音、スタンプ帳、賞状機能がある
- 子ども向け文言では、ふりがな表示に対応している

実装状況の詳細は `PROJECT_STATUS.md` を確認してください。

## 対象

- 小学1年生
- 小学2年生
- 小学3年生
- 家庭学習で算数を復習したい子ども
- 学習状況や苦手分野を確認したい保護者

## 主な機能

コード上で確認できる主な機能は以下です。

- 学年別学習
- 全学年ミックス
- サクサク10問モード
- じっくり5問モード
- 文章題ステップ式
- 筆算ステップ式
- 時計・単位・図形・表グラフ問題
- 苦手問題の自動保存
- 苦手こくふくモード
- 学習履歴保存
- 保護者画面
- メモボード
- 効果音オン・オフ
- スタンプ帳
- 賞状機能
- ふりがな表示

すべての機能の詳細な動作確認状況は、`PROJECT_STATUS.md` と `TEST_CHECKLIST.md` を確認してください。

外部APIや課金サービスは使わず、ローカルで動く構成です。

## ローカル起動方法

初回は依存関係をインストールします。

```bash
npm install
```

開発サーバーを起動します。

```bash
npm run dev
```

起動後、表示されたローカルURLをブラウザで開いて確認してください。

## ビルド確認

デプロイ前や作業後は、必ずビルドを確認してください。

```bash
npm run build
```

ビルド成果物は `dist/` に出力されます。

ローカルでビルド後の表示を確認する場合は、以下を使います。

```bash
npm run preview
```

## 手動テスト

手動確認には `TEST_CHECKLIST.md` を使ってください。

最低限、作業後は以下を確認することを推奨します。

- `npm run build` が成功する
- ホーム画面が表示される
- 学年を選べる
- 単元を選べる
- クイズを1問以上回答できる
- 結果画面に進める
- 苦手こくふく画面を開ける
- 保護者画面を開ける
- スマホ幅で大きく崩れていない

## AIで継続開発するとき

CodexなどのAIで作業する場合は、作業前に以下を読んでください。

- `AI_INSTRUCTIONS.md`
  - AI開発時のルール、設計方針、壊してはいけないもの
- `PROJECT_STATUS.md`
  - 現在の実装状況、要確認事項、次にやる候補
- `TEST_CHECKLIST.md`
  - 作業後に確認する手動テスト項目

これらのファイルは補助情報です。実装前には必ず現在のコードを確認し、ドキュメントと実際のコードに矛盾がある場合は現在のコードを優先してください。

## デプロイ前チェック

Netlifyなどの静的ホスティングへデプロイする前に、以下を確認してください。

- `npm install` が成功する
- `npm run build` が成功する
- `dist/` が生成される
- ホーム画面から主要画面へ移動できる
- 1年生・2年生・3年生のクイズを開始できる
- 結果画面へ進める
- localStorageを使う機能が落ちない
- 保護者画面が空データでも落ちない
- スマホ幅で大きな横はみ出しがない

Netlifyで使う場合の目安:

- Build command: `npm run build`
- Publish directory: `dist`

## 注意点

- localStorageに学習履歴、苦手問題、スタンプ、賞状、設定を保存します。
- 保存形式を変更する場合は、既存データとの互換性に注意してください。
- 子ども向け画面では、学年に合わせた文言とふりがな表示を大切にしてください。
- 新しいライブラリや外部サービスを追加する場合は、必要性を確認してから行ってください。
